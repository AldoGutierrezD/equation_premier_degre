Instructions

  • Demande à l'utilisateur de saisir les nombres aa, bb et cc.
  • Affiche l'équation ax+b=c ax+b=c dont l'utilisateur a saisi les coefficients.
  • Calcule puis affiche la solution ss de l'équation.

Rappel mathématique

ax+b=c ⟹ x=c-b/a

Exemple

Données
  • a = 3
  • b = 5
  • c = 17
Remplacement

3x+5=17 = 3x+5=17

Solution

s=(17-5)/3=4 = s=(17-5)/3=4.
